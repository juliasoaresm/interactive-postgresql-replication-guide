Guia de Replicação Streaming do PostgreSQL no Linux

Requisitos

  • PostgreSQL 15 instalado
  • Servidor primário (ex: 10.8.0.5)
  • Servidor réplica (ex: 10.8.0.7 ou 10.5.0.116)
  • Usuário replicador com permissão

Passo 1: Configurar a primária

  1. Ative o modo de arquivamento: 
    
archive_mode = on  
archive_command = 'cp %p /var/lib/postgresql/15/main/pg_wal_archive/%f'
  2. Ative a escuta e a conexão: 
    
listen_addresses = '*'
wal_level = replica
max_wal_senders = 10
  3. No pg_hba.conf da primária: 
    
host replication replicador 10.8.0.0/16 md5

Passo 2: Preparar a réplica

  1. Pare o serviço do PostgreSQL na réplica: 
    sudo systemctl stop postgresql
  2. Faça o backup com pg_basebackup: 
    
pg_basebackup -h 10.8.0.5 -U replicador -D /var/lib/postgresql/15/main -Fp -Xs -P
  3. Crie o arquivo standby.signal e configure o postgresql.auto.conf: 
    
primary_conninfo = 'host=10.8.0.5 user=replicador password=SUASENHA'
  4. Inicie o PostgreSQL: 
    sudo systemctl start postgresql
